# 999. Sort

*   https://www.toptal.com/developers/sorting-algorithms

## Overview

*   Insertion Sort
    *   ほとんど整列済みのデータに対しては高速
    *   オーバーヘッドが少ないので、小さいデータのソートに対しては有効
*   Selection Sort
    *   swap 回数を減らしたいとき
*   Bubble Sort
    *   Insertion Sort に近い
*   Shell Sort
    *   insersion sort の variant 的なもの
*   Merge Sort
    *   Stable
    *   O(n) space が必要
*   Heap Sort
    *   Not stable
*   Quick Sort
*   Bucket Sort
    *   重複が多いデータであれば高速
